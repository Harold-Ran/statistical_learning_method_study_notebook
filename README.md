# 《统计学习方法（第2版）》学习笔记

李航老师的《统计学习方法》是机器学习领域的经典入门教材之一。

本书包括监督学习和无监督学习两篇，全面系统地介绍了当前统计学习的经典方法，内容详实，公式推导详细。对于初学机器学习的小白来说，要完全理解这本书可能仍然有些困难，因此我们希望通过学习笔记的形式，以浅显直白的语言写出自己对书中章节的理解，供初入门的同学学习参考，有错漏之处也欢迎同学们帮助修正完善。
